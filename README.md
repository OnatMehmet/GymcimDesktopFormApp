# GymcimDesktopFormApp

C# DevFramework (N-Layer Architecture) - Gym management desktop (Winform .NET Framework) application.

# Features
- MS SQL Database
- Role based authentication
- Event Log

## Packages
- Entity Framework ORM
- Fluent Validation
- Metro Moden UI
- Ninject (Dependency Injector)
